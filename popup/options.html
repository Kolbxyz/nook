Nook Desktop

Hi there! Google Chrome is releasing a new version of the extension manifest which makes extensions like Nook impossible/very inconvenient to use.

Once Chrome updates to the new manifest version, this extension will not work and be unsupported. This version will take effect beginning in 2023. I've created a desktop version of this app with a bunch of new features, so if you like using Nook, you can install it! Feel free to remove this version when you have it.

I've added the following to the desktop version:

  • New slick interface
  • New rain sounds (in-game, no-thunder)
  • Town tunes (and customization)
  • Multilingual support (EN and ES for now)
  • Offline mode
  • Population growing snowy and cherry blossom themes
  • New horizons rainy and snowy themes
  • Pocket camp themes
  • Random mode
  • All K.K. songs released
